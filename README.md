# Kufar Bot — поиск телефонов и макбуков в Telegram

Бот раз в 5 минут (настраивается) проверяет Kufar.by и присылает вам новые
объявления по телефонам и макбукам, которые попадают в заданный диапазон
цены и (опционально) содержат ключевые слова в заголовке.

## Установка

1. Установите Python 3.9+.
2. Установите зависимости:

   ```
   pip install -r requirements.txt
   ```

3. Получите токен бота у [@BotFather](https://t.me/BotFather) в Telegram
   (команда `/newbot`).

4. Задайте токен переменной окружения и запустите бота:

   **Linux / macOS:**
   ```
   export BOT_TOKEN="ваш_токен_от_botfather"
   python3 bot.py
   ```

   **Windows (PowerShell):**
   ```
   $env:BOT_TOKEN="ваш_токен_от_botfather"
   python bot.py
   ```

Бот будет работать, пока запущен процесс. Чтобы он работал постоянно —
разместите его на сервере / VPS (например, через `systemd`, `pm2`, `screen`
или `tmux`) или на бесплатном хостинге вроде Railway / Render / Fly.io.

## Команды в Telegram

- `/start` — приветствие и инструкция.
- `/phone мин_цена макс_цена [ключевые слова]` — настроить поиск телефонов.
  Пример: `/phone 200 500 iphone 13` — телефоны от 200 до 500 долларов, в
  заголовке которых встречается "iphone" или "13".
- `/macbook мин_цена макс_цена [ключевые слова]` — настроить поиск макбуков.
  Пример: `/macbook 600 1200 air m1`.
- `/status` — показать текущие настройки.
- `/check` — проверить объявления немедленно, не дожидаясь автопроверки.
- `/stop_phone` — выключить поиск телефонов.
- `/stop_macbook` — выключить поиск макбуков.

Цены указываются в долларах США — Kufar отдаёт цену и в BYN, и в USD, бот
сравнивает по USD.

## Как это устроено

- `kufar_api.py` — обращение к неофициальному API Kufar.by
  (`api.kufar.by/search-api`) и разбор ответа.
- `storage.py` — простое хранилище на JSON-файле (`data.json`, создаётся
  автоматически): фильтры каждого пользователя и уже показанные объявления,
  чтобы не дублировать уведомления.
- `bot.py` — сам бот: команды, фоновый цикл проверки, отправка сообщений.

## Настройка интервала проверки

По умолчанию бот проверяет Kufar каждые 300 секунд (5 минут). Чтобы
изменить, задайте переменную окружения `CHECK_INTERVAL_SECONDS`, например:

```
export CHECK_INTERVAL_SECONDS=120
```

## Важно знать

- API Kufar неофициальный (сайт не публикует официальную документацию для
  сторонних разработчиков), поэтому Kufar может в любой момент изменить
  формат ответа — тогда потребуется поправить `kufar_api.py`.
- Категория «Ноутбуки» на Kufar общая для всех ноутбуков, поэтому поиск
  макбуков дополнительно фильтрует по слову "macbook" в запросе к Kufar и
  по вашим ключевым словам — на всякий случай можно уточнить модель
  (например, `/macbook 700 1500 air`).
- Бот хранит данные локально в файле `data.json` рядом с `bot.py` — не
  удаляйте его, иначе бот "забудет" все настройки и уже показанные
  объявления (и один раз пришлёт все объявления заново).
